Data Connectors (LlamaHub)#
Concept#
A data connector (aka
Reader) ingest data from different data sources and data formats into a simple
Document representation (text and simple metadata).
Tip
Once youâ€™ve ingested your data, you can build an Index on top, ask questions using a Query Engine, and have a conversation using a Chat Engine.

Our data connectors are offered through LlamaHub ðŸ¦™. LlamaHub is an open-source repository containing data loaders that you can easily plug and play into any LlamaIndex application.
Usage Pattern#
Get started with:
from llama_index.core import download_loader
from llama_index.readers.google import GoogleDocsReader
loader = GoogleDocsReader()
documents = loader.load_data(document_ids=[...])
